# 🎮 暴打三兄弟

一个真实办公室版的打击游戏！三兄弟会从工位后面冒出来，拿大锤子敲他们！

## 🎯 游戏玩法

- **目标**：在30秒内尽可能多地敲击弹出的"三兄弟"
- **计分**：每次敲击基础分10分，加上连击奖励
- **连击**：连续敲击可以获得加倍分数
- **背景**：真实的办公室场景设计

## 🎮 玩游戏

在线玩这个游戏：[点击这里](https://lvsiiiyiii66.github.io/beat-three-bros/)

或者在本地运行：
1. Clone 这个仓库
2. 用浏览器打开 `index.html` 文件
3. 开始暴打！

## 🎨 技术特点

- 纯 HTML5 + CSS3 + JavaScript
- 响应式设计，支持移动端
- 逼真的办公室场景渲染
- 流畅的动画效果
- 触摸和鼠标支持
